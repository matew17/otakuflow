# Otakuflow

This is an app dedicated to experiment and lear both VueJS, but also Agentic architecture using claude code.

## Project Setup

```sh
pnpm install
```

### Compile and Hot-Reload for Development

```sh
pnpm dev
```

### Type-Check, Compile and Minify for Production

```sh
pnpm build
```

### Run Unit Tests with [Vitest](https://vitest.dev/)

```sh
pnpm test
```

### Lint with [ESLint](https://eslint.org/)

```sh
pnpm lint
```

### Run e2e Tests with [Vitest](https://vitest.dev/)

```sh
pnpm test:e2e
```
